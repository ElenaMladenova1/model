    Master/detail

    When a document has identical fields in its header and lines, these fields are called master/detail attributes. They interact in a specific manner and share common principles and rules. For more information, see Master/Detail Attributes.

    To make sure the main principles are followed, there are business rules apply to each set of Master/Detail fields. Their action is decribed in the main topic Master/Detail Attributes.
