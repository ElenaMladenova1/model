    Table Gen_Document_Type_User_Statuses

    User-defined document statuses. Entity: Gen_Document_Type_User_Statuses

    Owner Tables Hierarchy

    • Gen_Document_Types

    Summary

    Name Type Description
    User_Status_Id uniqueidentifier PK
    Document_Type_Id uniqueidentifier The document type, to which this user status is bound
    State smallint Allowed: 10, 20, 30, 40, 50 The system state to which this user status is bound
    User_Status_Name nvarchar(128) Multi-language name of the user status
    Display_Order int Consecutive display order of the status, with regard to other statuses within the same document type
    Active bit Is the user status active for applying to documents
    Instructions nvarchar(2147483647) Instructions what should be done when this user status is active (Rich Text)
    Is_Exit_Status bit 1 when the status allows going to the next system status, 0 otherwise. For example for system status Firm Planned, only user status Approved might allow going to Released
    Row_Version timestamp
