    Table Crm_Customers

    Customer contracts list. For each combination of Enterprise Company and external Party there can be zero or one records of this. Entity: Crm_Customers

    Owner Tables Hierarchy

    • Gen_Parties

    Summary

    Name Type Description
    Customer_Id uniqueidentifier PK Unique customer Id
    Party_Id uniqueidentifier Base party Id
    Enterprise_Company_Id uniqueidentifier The Enterprise Company for which this customer is recorded. The same external party can be listed with different conditions for the different enterprise companies.
    From_Date datetime Start date of the customer relationship
    Thru_Date datetime The date of customer relationship termination. NULL for active customers.
    Default_Price_List_Id uniqueidentifier If not NULL, specifies default price list when selling to this customer
    Default_Payment_Term_Days int Default payment term in days when issuing documents for this customer
    Default_Delivery_Term_Days int Default term in days for goods delivery, starting at the day of sale
    Customer_Number nvarchar(16) Unique customer number
    Default_Payment_Type_Id uniqueidentifier If not NULL, specifies default payment type for the sales, offers and invoices for this customer
    Credit_Limit decimal(18, 2) Total credit limit for the customer in the customers' default currency. NULL means there is no limit
    Sales_Person_Id uniqueidentifier The default sales person for new sales documents for this customer
    Creation_Time datetime Readonly
    Creation_User nvarchar(64) Readonly
    Update_Time datetime Readonly
    Update_User nvarchar(64) Readonly
    Collections_Responsible_Employee_Id uniqueidentifier The employee, who is responsible for the collections from the customer
    Default_Payment_Account_Id uniqueidentifier The default payment account to use when creating new documents for this customer
    Default_Currency_Id uniqueidentifier The primary currency for value calculations for this customer - for credit limit, due amounts, etc.
    Customer_Type_Id uniqueidentifier The user-defined type of this customer. NULL when there is no specific type. Record-level security from the customer type is applied to the individual customers and can be used for security purposes
    Persist_Sales_Orders_Lots bit If checked, specifies that the lots set in the Sales orders for this customer, cannot be changed during the execution of the Store transactions for these Sales orders.
    Default_Distribution_Channel_Id uniqueidentifier The default distribution channel used when selling to the customer
    Active bit 1 if the customer is active, 0 - not to list in combo boxes for choosing in new documents
    Default_Payment_Start_Days int Specifies the number of days after the sales order, when the payment becomes due. 0 means that the payment is due immediately.
    Grace_Period_Days int Number of days after the payment deadline, during which the system still allows new sales orders for the customer.
    Serviced_By_Enterprise_Company_Location_Id uniqueidentifier The enterprise company location, which sells to this client by default.
    Allow_Use_As_Primary_Customer bit Specifies whether to allow the customer to be used as primary customer in a sales deal.
    Allow_Use_As_Ship_To_Customer bit True to allow the customer to be used as ship to customer in a sales deal.
    Row_Version timestamp
