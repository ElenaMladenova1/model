    Procedure Acc_Templates_Table_Get_Template_Lines

    Summary

    Name Type Description
    Template_Line_Id uniqueidentifier PK
    Template_Id uniqueidentifier
    Line_No int Consecutive number of the line within the template. Determines the order of execution of the template lines.
    Filter_XML nvarchar The line is accounted only when the filter is satisfied compared with the source document header
    Debit_Account_Id uniqueidentifier The account which should be debited
    Credit_Account_Id uniqueidentifier The account which should be credited
    Amount_Source_Name nvarchar The source rowset for the amount. For example: DocLines, DocHeader, Additional Amounts, Stock Types, etc.
    Amount_Row_Id uniqueidentifier The id of the row from the amount rowset where the amount is located. NULL means to account one by one for all rows within the rowset
    Amount_Row_Name nvarchar The name of definition, specified in Amount_Row_Id. NULL means that no Amount_Row_Id is specified or there is no name
    Amount_Source_Filter nvarchar Filter that further specifies which rows from the amount rowset determine the amount
    Amount_Column_Name nvarchar The name of the column within the amount rowset where the amount is located
    Multiplier decimal(0, 0)
    Notes nvarchar
    Amount_Condition nvarchar
    Valid_From_Date datetime
    Valid_To_Date datetime
    Row_Version timestamp
