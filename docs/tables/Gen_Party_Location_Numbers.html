    Table Gen_Party_Location_Numbers

    Location numbers for a party. Depending on the partner with which we are doing an exchange, our location number might be different. Entity: Gen_Party_Location_Numbers (Introduced in version 19.1)

    Owner Tables Hierarchy

    • Gen_Parties

    Summary

    Name Type Description
    Party_Location_Number_Id uniqueidentifier PK
    Party_Id uniqueidentifier The party for which we are defining the location number.
    Partner_Party_Id uniqueidentifier The party with which we are doing exchange. Depending on the Partner Party, the main Party might have different location number. NULL means that the location number is not dependent on the Partner Party.
    Partner_Location_Number nvarchar(16) The location number of the partner party for which we define the main Party location number. The location number of the main Party might be different depending on the location number of the partner party. NULL means that the location number is not dependent on the partner location number.
    Location_Coding_System nvarchar(3) Allowed: GLN, INT The coding system for which we are defining the location number.
    Location_Number nvarchar(16) The location number of Party.
    Significance int Order of significance of the location number within the main Party. If there are multiple location numbers, only the most significant is used. 0 is the least significant and higher numbers indicate higher significance.
    Row_Version timestamp
