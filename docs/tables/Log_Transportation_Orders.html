    Table Log_Transportation_Orders

    Transportation ordered to specific carrier or vehicle. Entity: Log_Transportation_Orders

    Owner Tables Hierarchy

    • Gen_Documents

    Summary

    Name Type Description
    Transportation_Order_Id uniqueidentifier PK
    Document_Id uniqueidentifier
    Departure_Date date Planned departure date.
    Departure_Time time Planned departure time.
    Arrival_Date date Planned arrival date.
    Arrival_Time time Planned arrival time.
    Transportation_Mode_Id uniqueidentifier The mode of transportation which should be used for this order. NULL when it is unknown or doesn't matter.
    Carrier_Id uniqueidentifier The carrier chosen for this transportation. NULL when the transportation is done with own transport.
    Transportation_Vehicle_Id uniqueidentifier The own transportation vehicle, which will be used for the transportation.
    From_Geo_Point_Id uniqueidentifier The geographic point from which the route begins.
    To_Geo_Point_Id uniqueidentifier The geographic point at which the route ends.
    Row_Version timestamp
    Crew_Id uniqueidentifier The crew which is assigned to operate the vehicle of the current Transportation Order. If null means the crew is yet unknown or the order is executed by external carrier.
