    Table Gen_Document_Jobs

    Represents jobs, which change state of documents. Entity: Gen_Document_Jobs (Introduced in version 20.1)

    Owner Tables Hierarchy

    • Sys_Jobs

    Summary

    Name Type Description
    Document_Job_Id uniqueidentifier PK
    Job_Id uniqueidentifier The id of the general job.
    Document_Type_Id uniqueidentifier The type of the document, whose state will be changed.
    Filter_Xml nvarchar(2147483647) Specifies the documents, whose state will be changed.
    New_State smallint Allowed: 0, 10, 20, 30, 40, 50, 5 Specifies the state, which should be set to the documents, matching the filter.
    Row_Version timestamp
    New_User_Status_Id uniqueidentifier Specifies the user status, which should be set to the documents, matching the filter. The user statuses are defined for the selected document type.
