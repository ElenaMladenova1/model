    Table Sys_External_Applications

    List of external applications. Used to run external applications. Entity: Sys_External_Applications

    Summary

    Name Type Description
    External_Application_Id uniqueidentifier PK
    Entity_Name nvarchar(64) The entity, for which the application is defined.
    External_Application_Name nvarchar(254) ML The name of the external application.
    Platform nvarchar(12) Allowed: WINDOWS, WEBPAGE, WEBSERVICE, ANDROID, iOS The execution platform of the application.
    Address nvarchar(2147483647) The address (path) to the application. The address is platform-dependant.
    Available_In_Mode nvarchar(6) Allowed: SINGLE, LIST, ALL Whether the application will be displayed to the end users when a single object is opened or when multiple objects are listed.
    Execute_For_All_Objects bit Whether to execute the application for all selected objects at once or to execute the application for each object, one by one.
    Save_Before_Start bit Whether to save the form data to the server before starting the application.
    Refresh_After_Finish bit Whether to refresh the data, displayed to the user, after the execution finishes. Not all platforms and/or applications support finish notification.
    Notes nvarchar(2147483647)
    Access_Key_Id uniqueidentifier
    Is_Active bit Specifies whether the external application is currently used.
    Row_Version timestamp
