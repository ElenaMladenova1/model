    Table Llm_Model_Builds

    Entity

    Entity: Projects.AI.ModelBuilds

    Successive builds of sub-models, containing the QAs available at the time of the build. Created automatically by the Build now function in the model. Entity: Llm_Model_Builds (Introduced in version 24.1.1.93)

    Owner Tables Hierarchy

    • Llm_Models

    Summary

    Name Type Description
    Build_Completion_Time_Utc datetime The time when the build was completed.
    Build_Log nvarchar(max) Build log, as returned by the model provider.
    Build_Start_Time_Utc datetime The time when the build job was started.
    Build_User_Id uniqueidentifier The user which started the build.
    Model_Build_Id uniqueidentifier PK
    Model_Id uniqueidentifier
    Result_Model_Id uniqueidentifier The name of the model, which was created by the build. Updated after the build job completes.
    Row_Version timestamp
    Status nvarchar(1) Allowed: N, R, C Indicates the status of the build - New, Running or Completed.
    Success bit Indicates whether the build was successfully built.
