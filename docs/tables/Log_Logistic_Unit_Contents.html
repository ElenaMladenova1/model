    Table Log_Logistic_Unit_Contents

    Theoretical or actual content of a logistic unit. Entity: Log_Logistic_Unit_Contents (Introduced in version 21.1.0.77)

    Owner Tables Hierarchy

    Summary

    Name Type Description

    Logistic_Unit_Id uniqueidentifier The containing logistic unit.
    Line_No int Consecutive position within the logistic unit.
    Product_Id uniqueidentifier The product, which is contained in the logistic unit.
    Quantity decimal(12, 3) Quantity of the product in the logistic unit. Expressed in the specified measurement unit.
    Quantity_Unit_Id uniqueidentifier The measurement unit of the quantity.
    Standard_Quantity decimal(12, 3) The quantity, expessed in the standard measurement unit of the product.
    Base_Quantity decimal(12, 3) The quantity, expressed in the base measurement category of the product.
    Notes nvarchar(2147483647)
    Gross_Weight decimal(12, 3) Gross weight in kilograms (kg). NULL means unknown.
    Lot_Number nvarchar(32) The production lot number. NULL means unknown.
    Expiration_Date date Expiration date of the goods. NULL means unknown or N/A.
