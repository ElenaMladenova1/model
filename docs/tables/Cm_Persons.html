    Table Cm_Persons

    Personal records. Requires related party record. Entity: Cm_Persons

    Owner Tables Hierarchy

    • Gen_Parties

    Summary

    Name Type Description
    Person_Id uniqueidentifier PK Unique person Id
    First_Name nvarchar(64) ML First name of the person.
    Middle_Name nvarchar(64) ML Middle name of the person.
    Last_Name nvarchar(64) ML Last name of the person.
    Title nvarchar(32) ML The persons title. It may be one of the standard titles - Mr., Mrs., etc., but it can also contains professional or academic qualification.
    Gender nvarchar(1) Allowed: F, M Person gender. M=Male;F=Female;NULL=not known/not provided
    Birth_Date datetime Birth date. NULL means unknown
    Nationality uniqueidentifier Person's nationality. NULL means the default (same as enterprise) nationality
    National_Number nvarchar(14) Government assigned unique personal number. NULL means unknown
    Passport_Number nvarchar(14) Current passport number. NULL means unknown
    Passport_Issuing_Date datetime Date of issuing the passport. NULL means unknown
    City nvarchar(64) ML City from the legal registration address for the person
    Notes nvarchar(254) Internal notes for the person
    Update_User nvarchar(64) Readonly
    Creation_Time datetime Readonly
    Creation_User nvarchar(64) Readonly
    Update_Time datetime Readonly
    Party_Id uniqueidentifier The Id of the party record for this person. The party stores the person's full name
    Row_Version timestamp
