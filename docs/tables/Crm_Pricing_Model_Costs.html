    Table Crm_Pricing_Model_Costs

    List of costs, associated with a pricing model. Entity: Crm_Pricing_Model_Costs

    Owner Tables Hierarchy

    Summary

    Name Type Description
    Pricing_Model_Cost_Id uniqueidentifier PK Pricing Model Cost
    Pricing_Model_Id uniqueidentifier Pricing model, for which the cost is defined
    Line_No int The consequtive number of the cost within the pricing model
    Cost_Type_Id uniqueidentifier The cost type, which is listed for the pricing model
    Amount_Percent decimal(6, 5) Used when the cost is calculated as percent of the amount. NULL when the cost is calculated in a different way
    Amount_Per_Unit decimal(9, 2) Used when the cost is calculated as amount per unit of the product. The unit is defined in Measurement_Unit_Id. NULL when the cost is calculated in a different way
    Measurement_Unit_Id uniqueidentifier The measurement unit for which the Amount_Per_Unit is defined. NULL when the cost is calculated in a different way
    Row_Version timestamp
