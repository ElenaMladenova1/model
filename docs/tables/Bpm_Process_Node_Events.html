    Table Bpm_Process_Node_Events

    Abstract root of all process node events. Currently - not used. Entity: Bpm_Process_Node_Events

    Summary

    Name Type Description
    Process_Node_Event_Id uniqueidentifier PK
    Process_Node_Id uniqueidentifier The node of this node event.
    Event_Type nvarchar(1) Event type. S=Start, T=Intermediate Throw, C=Intermediate Catch, B=Boundary, E=End.
    Is_Message bit True if this is message event.
    Is_Timer bit True if this is timer event.
    Is_Signal bit True if this is signal event.
    Is_Escalation bit True if this is escalation event.
    Is_Error bit True if this is error event.
    Is_Cancel bit True if this is cancel event.
    Is_Compensation bit True if this is compensation event.
    Event_Key nvarchar(128) The unique event key, which is thrown or caught.
    Boundary_Of_Process_Node_Id uniqueidentifier When the event is boundary, contains the node to which the event is bound. Otherwise contains NULL.
    Row_Version timestamp
