    Name Type Description
    Document_Id uniqueidentifier The task's document (for entry date, task type, status)
    System_Type nvarchar Allowed: C, M, T T=Task; C=Communication; M=Meeting
    Subject nvarchar Task primary subject (required)
    Activity_Notes nvarchar
    Target_Party_Id uniqueidentifier External participant or target of the task
    Responsible_Party_Id uniqueidentifier Who is responsible for executing the task. Initially this is the owner of the task
    Owner_Party_Id uniqueidentifier The party that owns the task. Initially this is the party that has created the task
    Priority tinyint Allowed: 1, 2, 3, 4, 5 Priority on the scale from 1 (least important) to 5 (very important)
    Private bit 1 if the task is visible only to its owner; 0 if this is publicly visible task
    Activity_Start_Time datetime Starting time of the task
    Deadline_Time datetime The deadline for the task. NULL if there is no deadline
    Activity_End_Time datetime The time when the task was completed. NULL if the task is not yet completed
    Reminder_Time datetime When to snooze to the owner to remind him for the task
    Contact_Person_Id uniqueidentifier
    Planned_Duration_Minutes int Total planned duration of the activity, regardless of the current execution status
    Project_Task_Id uniqueidentifier The project task for which the work is performed. NULL when the activity is not related to a project task.
    Is_Single_Execution bit Readonly Specifies whether the document is a single execution of its order document.
    Is_Released bit Readonly True if the document is not void and its state is released or greater
    From_Party_Id uniqueidentifier The party which issued the document
    Enterprise_Company_Id uniqueidentifier The enterprise company that is issuing the document. NULL means that there is only one such company in the database and the reporting is always summarized
    Document_Type_Id uniqueidentifier
    Document_No nvarchar Document number, unique within Document_Type_Id
    Document_Date date
    Reference_Date datetime The date to which this document refers, i.e. when the action really occurred. If NULL, Document_Date is taken
    Current_State smallint Allowed: 0, 5, 10, 20, 30, 40, 50, Readonly The current system state of the document. Allowed values: 0=New;5=Corrective;10=Computer Planned;20=Human Planned;30=Released;40=Completed;50=Closed.
    Void bit Readonly 1 if the document is null and void
    To_Party_Id uniqueidentifier The party which should receive the document
    Currency_Directory_Id uniqueidentifier
    Master_Document_Id uniqueidentifier In a multi-document tree, this is the root document, that created the whole tree. If this is the root it is equal to Id
    Parent_Document_Id uniqueidentifier In a multi-document tree, this is the direct parent document. If this is the root it is NULL
    Adjusted_Document_Id uniqueidentifier Readonly
    Adjustment_Number int Readonly Consecutive number of the correction that this document is applying to the adjusted document.
    Creation_Time datetime Readonly Date/Time when the document was created
    Creation_User nvarchar Readonly The login name of the user, who created the document
    Void_Time datetime Readonly Date/time when the document has become void
    Void_User nvarchar Readonly The user who voided the document
    Adjustment_Time datetime Readonly Date/time when the document last has been adjusted by corrective document
    Adjustment_User nvarchar Readonly The user who adjusted the document
    Read_Only bit Readonly 1 - the document is read only; 0 - the document is not read only
    Document_Notes nvarchar
    Document_Version int Readonly
    Responsible_Person_Id uniqueidentifier The person that is responsible for this order or transaction. It could be the sales person, the orderer, etc.
    Void_Reason nvarchar Readonly
    Reverse_Of_Document_Id uniqueidentifier Readonly The document which the current document is reverse of
    Enterprise_Company_Location_Id uniqueidentifier
    Planning_Only bit Readonly Indicates that the document is used only for planning (and as consequence its state cannot be greater than Planned)
    Prime_Cause_Document_Id uniqueidentifier The document that is the prime cause for creation of the current document
    Release_Time datetime Readonly
    Complete_Time datetime Readonly
    Parent_Document_Relationship_Type nvarchar Allowed: S, N, Readonly Type of relationship between the current document and the parent document(s). Affects the constraints for execution/completion for the documents. Possible values: 'S' = 'Subtask', 'N' = 'Next task'.
    Reference_Document_No nvarchar The number of the document (issued by the other party), which was the reason for the creation of the current document. The numebr should be unique within the party documents
    Current_User_Status_Id uniqueidentifier Readonly The user status of this document if applicable for this document type. NULL means unknown or not yet set
    From_Company_Division_Id uniqueidentifier The division of the company, issuing the document. NULL when the document is not issued by any specific division
    To_Company_Division_Id uniqueidentifier The division of the company, receiving the document. NULL when the document is not received by any specific division
    Entity_Name nvarchar The entity name of the document header.
    Activity_Time_Interval_Id uniqueidentifier PK The id of the time interval
    Activity_Id uniqueidentifier The activity for which the time interval is recorded
    Party_Id uniqueidentifier The party for which the time interval is recorded
    Date date The date on which the work was performed
    Start_Time time The starting time of the time interval within 'Date'
    End_Time time The ending time of the time interval within 'Date'
    Notes nvarchar Notes for the time interval
    Execution_Complete_Percent decimal(0, 0) Percent of task completed
    State smallint Allowed: 0, 5, 10, 20, 30, 40, 50 The state of the primary activity in the moment the time interval was created. NULL when the state is unknown.
    User_Status_Id uniqueidentifier The user status of the primary activity in the moment the time interval was created. NULL when the user status is unknown.
