    Table Acc_Cost_Centers

    Defines cost centers and their relation to profit centers. Entity: Acc_Cost_Centers

    Summary

    Name Type Description
    Cost_Center_Id uniqueidentifier PK
    Parent_Cost_Center_Id uniqueidentifier Parent cost center, NULL if this is root cost center
    Cost_Center_Name nvarchar(254)
    Relative_Weight decimal(18, 4) The weight of this cost center, relative to the other cost centers within the same parent. The weight is used during cost distribution calculations. The sum of weights of all cost centers within a parent does not need to be 100, the ratio of total weight over weight is used.
    Notes nvarchar(254)
    Transfer_Cost_To_Profit_Center_Id uniqueidentifier Points to the profit center to which to transfer the cost. Valid only for leaf cost centers
    Row_Version timestamp
