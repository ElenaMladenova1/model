    Table Gen_Document_Type_Properties

    Default user-defined properties, which should be added to new documents. Entity: Gen_Document_Type_Properties

    Owner Tables Hierarchy

    • Gen_Document_Types

    Summary

    Name Type Description
    User_Property_Id uniqueidentifier PK
    Document_Type_Id uniqueidentifier The document type, for which to add user-defined properties.
    Line_No int Line number, unique within the document type
    Property_Id uniqueidentifier The user-defined property, which should be added.
    Default_Property_Value nvarchar(254) The default value of the property when creating new documents.
    Default_Property_Value_Description nvarchar(254) ML Default description value of the property when creating new documents.
    Required bit 1 if the property is required when creating documents of this type
    Enterprise_Company_Id uniqueidentifier When not null, specifies that the current rule is valid only for the specified company.
    Required_From_Date datetime When not null, specifies a date, after which the property becomes required for the current document type. The date is compared against the document date.
    Required_Thru_Date datetime When not null, specifies a date, up to which the property is required for the current document type. The date is compared against the document date.
    Default_Value_Id uniqueidentifier Internal Id of the default value of the property.
    Row_Version timestamp
