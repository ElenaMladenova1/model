    Table Log_Transportation_Execution_Lines

    Contains details of executions of transportation order lines. Entity: Log_Transportation_Execution_Lines (Introduced in version 18.2)

    Owner Tables Hierarchy

    • Log_Transportation_Executions
    • Gen_Documents

    Summary

    Name Type Description
    Transportation_Execution_Line_Id uniqueidentifier PK
    Transportation_Execution_Id uniqueidentifier
    Line_No int Consecutive line number within this execution.
    Execution_Of_Transportation_Order_Line_Id uniqueidentifier The transportation order line, which is executed.
    Geo_Point_Id uniqueidentifier The geographic point, where the operation is executed.
    Operation_Type nvarchar(1) Allowed: L, U, O The type of operation being executed. L=Loading; U=Unloading; O=Other.
    Pallets_Count int Number of pallets affected by this operation. NULL when unknown or N/A.
    Weight_Kg int Cargo weight in kg, affected by this operation. NULL when unknown or N/A.
    Volume_Cbm int Cargo volume in cubic meters, affected by this operation. NULL when unknown or N/A.
    Execution_Date date The date when the operation was executed.
    Execution_Time time The time when the operation was executed.
    Pallet_Number nvarchar(32) Pallet number, when applicable. NULL when unknown or not applicable.
    Notes nvarchar(2147483647)
