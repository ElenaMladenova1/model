    Table Inv_Transfer_Orders

    Orders for stock transfers between stores. Entity: Inv_Transfer_Orders

    Owner Tables Hierarchy

    • Gen_Documents

    Summary

    Name Type Description
    Transfer_Order_Id uniqueidentifier PK
    Document_Id uniqueidentifier The unique stock transfer order id, as per documents table
    Default_Due_Date_Out datetime When the transfer is scheduled to issue the goods from the source warehouse
    Default_Due_Date_In datetime The date, when the goods are expected to be received in the destination warehouse
    From_Store_Id uniqueidentifier Specifies the store, from which the goods will be issued.
    To_Store_Id uniqueidentifier Specifies the store, in which the goods will be received.
    Notes nvarchar(2147483647)
    Status smallint Allowed: 1, 2 Obsolete. Not used. (Status of the transfer document: 1-Open for editing; 2-Released)
    Is_Single_Execution bit Readonly Specifies whether the document is a single execution of its order document.
    Is_Released bit Readonly True if the document is not void and its state is released or greater
    Row_Version timestamp
