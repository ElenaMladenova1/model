    Table Gen_Printouts

    Contains data about binding of printout layouts to specific user-defined document types. Entity: Gen_Printouts

    Owner Tables Hierarchy

    • Gen_Document_Types

    Summary

    Name Type Description
    Printout_Id uniqueidentifier PK
    Application_Name nvarchar(64) The application which stored and uses the printout
    Printout_Name nvarchar(64) The name of the printout. Unique within the application form
    Enterprise_Company_Id uniqueidentifier
    Document_Type_Id uniqueidentifier The document type to which this printout layout is bound.
    Printout_Layout_Id uniqueidentifier The printout layout, that is bound to the document type.
    Definition_Format nvarchar(16) Obsolete. Not used.
    Definition nvarchar(1073741823) Obsolete. Not used.
    Is_Default bit 1 if this is the default printout for the application form
    Ord int Order in the list of printouts when using direct printing
    Copies int Number of copies that should be printed when using direct printing
    Report_Id uniqueidentifier If not NULL points to a custom report that indicates which data will be loaded in the printout.
    Backward_Compatibility bit Obsolete. Not used.
    Allow_Printing_On_State smallint Allowed: 0, 10, 20, 30, 40, 50 The user can print documents only with state equal or greater than Allow_Printing_On_State
    Ord_Priority int Ordinal position and priority of the printout, in regard to other printouts within the current document type. Used for sorting, when executing printouts with "Print All" command.
    Ord_Filter_Xml nvarchar(2147483647) The condition, required to be matched in order for the printout to be executed upon "Print All" command.
    Notes nvarchar(512)
    Row_Version timestamp
