    Procedure Crm_Sales_Orders_Table_Printout_Lines

    Summary

    Name Type Description
    Line_No int Consecutive number of the line within the sales order
    Product_Id uniqueidentifier The product sold
    Product_Description nvarchar ML The name of the sold product at the time the sale was made
    Quantity decimal(12, 3) The quantity sold
    Quantity_Unit_Id uniqueidentifier The measurement unit of Quantity
    Unit_Price decimal(14, 5) Unit price of the product in the currency of the sales order and in the unit of measure, as specified by QuantityUnitId
    Line_Amount decimal(14, 2) The total amount for the line. Equals to Quantity * Unit_Price, less the discounts
    Line_Custom_Discount_Percent decimal(7, 6) User-defined discount for the line
    Line_Discount_Id uniqueidentifier The line discount type used to form the Line_Standard_Discount_Percent
    Line_Standard_Discount_Percent decimal(7, 6) Readonly Standard discount for the line. This is automatically computed according to discount conditions
    Product_Price_Id uniqueidentifier Not NULL when the price has been selected from the list of valid standard prices
    Required_Delivery_Date date The required (contracted) delivery date for the line
    Guarantee_Period_Days int Guarantee period in days for the offered product. NULL for non-serviced products
    Sales_Order_Line_Id uniqueidentifier PK
    Sales_Order_Id uniqueidentifier
    Line_Deal_Type_Id uniqueidentifier Deal type to be passed to the invoice line. If deal type in entered then the invoice creates VAT entry for this deal type.
    Quantity_Unit_Id_Text nvarchar ML Name of the measurement unit
    Quantity_Base decimal(0, 0) Readonly
    Product_Name nvarchar ML Name of the item
    Product_Id_Text nvarchar ML Name of the item
    Line_Discount_Id_Text nvarchar
    Product_Price_Id_Text decimal(0, 0)
    Return_For_Sales_Order_Line_Id uniqueidentifier
