    Query Cms_News_Web_Modules_Table_Load_Navigator

    Summary

    Name Type Description
    News_Web_Module_Id uniqueidentifier PK
    Web_Module_Id uniqueidentifier The web module, which this News Module inherits.
    Web_Site_Id uniqueidentifier The web site to which the module belongs.
    Parent_Web_Module_Id uniqueidentifier The parent web Module. It can only be a module with Module Type = Category. Null means that the web Module is root web Module.
    Web_Module_Name nvarchar Multilanguage Module name. This is used as a title when displaying the web Module.
    Local_Url nvarchar The local Module Url. The full Url of a module is formed based on the sites Base Url, the local Urls of all parent modules and the module Local Url. Null means that the web Module is the default web module within the parent path.
    Is_Published bit Specifies whether the module is published and will be showed in the web site.
    Module_Picture varbinary Default picture. Used for picture links, small module icons, etc. PNG format is suggested.
    Module_Type nvarchar Allowed: CAT, STA, NEW Specifies the content handler. The content handler is responsible for generating the actual web page content. Module Types include Category, Static, News, Product, etc.
