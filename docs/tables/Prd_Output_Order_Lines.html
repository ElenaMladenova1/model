    Table Prd_Output_Order_Lines

    Detail records of output orders. Entity: Prd_Output_Order_Lines

    Owner Tables Hierarchy

    • Prd_Output_Orders
    • Gen_Documents

    Summary

    Name Type Description
    Id uniqueidentifier PK
    Output_Order_Id uniqueidentifier
    Line_Ord int Line number within the order
    Line_Work_Order_Id uniqueidentifier The work order for which work is being accounted
    Work_Order_Item_Id uniqueidentifier The work order item for which this output is recorded.
    Work_Order_Item_Operation_Id uniqueidentifier The operation for which this output is recorded.
    Transaction_Timestamp datetime Readonly Sets the timestamp of the receipt store operations for this output order line. Used in completing output orders.
    Product_Code_Id uniqueidentifier Selects the product thru some of the product codes.
    Lot_Id uniqueidentifier The lot of the produced product.
    Workgroup_Resource_Id uniqueidentifier The resource, which was used to perform the operation.
    Work_Done_By_Party_Id uniqueidentifier The party (usually employee worker) who has accomplished the work. NULL if unknown or N/A
    Notes nvarchar(2147483647)
    End_Time datetime Date and time when the operation has ended
    Start_Time datetime Date and time when the operation has began
    Store_Bin_Id uniqueidentifier If not NULL, specifies that the product was (has to be) stored to specific store bin
    Serial_Number_Id uniqueidentifier If not NULL, specifies that the product was (has to be) stored with specific serial number
    Row_Version timestamp
    Store_Id uniqueidentifier Where to output the produced quantity. Can be NULL only if Produced_Quantity = 0
    Produced_Quantity_Base decimal(18, 3) Readonly The equivalence of Produced Quantity in the base measurement unit of the product.
    Produced_Quantity_Unit_Id uniqueidentifier The measurement unit of Produced Quantity.
    Produced_Quantity decimal(18, 3) The processed quantity of the end product.
    Begin_Date_Time datetime Not used.
    Product_Id uniqueidentifier The actually produced product
    Finished bit 1 if this output entry completes the operation. 0 if there might be more entries
    Produced_Standard_Quantity_Base decimal(18, 3) Readonly The theoretical quantity in base measurement unit according to the current measurement dimensions for the product. Used to measure the execution. NULL means to convert the value from Quantity using the measurement ratios.
