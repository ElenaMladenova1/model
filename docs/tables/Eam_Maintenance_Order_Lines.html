    Table Eam_Maintenance_Order_Lines

    Contains the types of maintenance and maintained assets in the maintenance orders. Entity: Eam_Maintenance_Order_Lines (Introduced in version 19.1)

    Owner Tables Hierarchy

    • Eam_Maintenance_Orders
    • Cm_Activities
    • Gen_Documents

    Summary

    Name Type Description
    Line_No int Consecutive line number, unique within the maintenance order.
    Maintenance_Order_Line_Id uniqueidentifier PK
    Maintenance_Order_Id uniqueidentifier
    Managed_Asset_Id uniqueidentifier The maintained asset.
    Maintenance_Type_Id uniqueidentifier The type of maintenance performed.
    Next_Service_Date date Specifies, that the maintenance required a specific date for the next maintenance. NULL means that default scheduling should be used.
    Next_Service_Tracked_Parameter_Value int Specifies, that the maintenance required the next maintenance to be performed on a specific value of the tracked parameter. NULL means that default scheduling should be used.
    Notes nvarchar(2147483647)
