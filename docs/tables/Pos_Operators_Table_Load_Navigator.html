    Query Pos_Operators_Table_Load_Navigator

    Summary

    Name Type Description
    Pos_Operator_Id uniqueidentifier PK
    Pos_Location_Id uniqueidentifier The POS location where this operator works.
    Pos_Operator_Code nvarchar Operator code. Unique within the Pos Location.
    User_Id uniqueidentifier The login user for this POS operator.
    Pos_Role_Id uniqueidentifier The role, assigned to the operator. The role indicates the permissions of the operator for this POS location.
    Default_Pos_Terminal_Id uniqueidentifier The default POS terminal for this opertor. NULL when there is no default.
    Starting_Date date The first date, when the operator has started working for this POS location.
    Termination_Date date The date, when the operator has ceased working in this POS location. NULL means, that the operator is still working or the termination date is still unknown.
    Is_Active bit Indicates whether this operator is active and can be chosen for new records.
    Enterprise_Company_Id uniqueidentifier
    Enterprise_Company_Location_Id uniqueidentifier The enterprise company location of the POS location. Currently, only one POS location is allowed for each company location.
    Pos_Location_Code nvarchar Unique (with the enterprise company) code of this POS location.
    Pos_Location_Is_Active bit Indicates whether the POS location is currently active and can be chosen in drop-downs, etc.
    Pos_Role_Code nvarchar Unique role code.
    Pos_Role_Name nvarchar ML Multi-language name of the POS role.
    Can_Void_Sales bit Indicates whether this role can void sales orders.
    Can_Process_Minus_Sales bit Indicates whether the role is allowed to process minus (qty and/or value) sales.
    Person_Id uniqueidentifier PK Unique person Id
    Party_Id uniqueidentifier The Id of the party record for this person. The party stores the person's full name
    First_Name nvarchar ML First name of the person.
    Middle_Name nvarchar ML Middle name of the person.
    Last_Name nvarchar ML Last name of the person.
    Title nvarchar ML The persons title. It may be one of the standard titles - Mr., Mrs., etc., but it can also contains professional or academic qualification.
    Gender nvarchar Allowed: F, M Person gender. M=Male;F=Female;NULL=not known/not provided
    Birth_Date datetime Birth date. NULL means unknown
    Nationality uniqueidentifier Person's nationality. NULL means the default (same as enterprise) nationality
    National_Number nvarchar Government assigned unique personal number. NULL means unknown
    Passport_Number nvarchar Current passport number. NULL means unknown
    Passport_Issuing_Date datetime Date of issuing the passport. NULL means unknown
    City nvarchar ML City from the legal registration address for the person
    Notes nvarchar Internal notes for the person
    Creation_Time datetime Readonly
    Creation_User nvarchar Readonly
    Update_Time datetime Readonly
    Update_User nvarchar Readonly
