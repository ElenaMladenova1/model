    Birth_Date datetime Birth date. NULL means unknown
    Nationality uniqueidentifier Person's nationality. NULL means the default (same as enterprise) nationality
    National_Number nvarchar Government assigned unique personal number. NULL means unknown
    Passport_Number nvarchar Current passport number. NULL means unknown
