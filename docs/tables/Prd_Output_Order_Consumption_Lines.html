    Table Prd_Output_Order_Consumption_Lines

    The internal production consumption of materials. Used only for control purposes - for comparison with the active consumption orders. Warehouse and accounting are still updated only by Consumption Orders. Entity: Prd_Output_Order_Consumption_Lines

    Owner Tables Hierarchy

    • Prd_Output_Orders
    • Gen_Documents

    Summary

    Name Type Description
    Output_Order_Consumption_Line_Id uniqueidentifier PK
    Output_Order_Id uniqueidentifier
    Line_No int
    Line_Work_Order_Id uniqueidentifier Work order for the line. Initially copied from the header (if there work order is not null)
    Work_Order_Item_Id uniqueidentifier The work order item, for which the consumption is recorded.
    Work_Order_Item_Operation_Id uniqueidentifier The operation in which the material was consumed. NULL = unknown
    Material_Id uniqueidentifier The consumed material
    Consumed_Quantity_For_Setup decimal(18, 3) The consumed quantity of the material while setting up the operation.
    Consumed_Quantity_For_Run decimal(18, 3) The consumed quantity of the material for the operation.
    Consumed_Quantity_For_Scrap decimal(18, 3) The scrapped quantity of the material for this operation.
    Quantity_Unit_Id uniqueidentifier The measurement unit for the consumed quantity for setup, run and scrap.
    Scrap_Type_Id uniqueidentifier When Consumed_Quantity_For_Scrap <> 0 denotes the reason for the scrap
