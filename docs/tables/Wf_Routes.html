    Table Wf_Routes

    Contains document routes, which specify which document generation procedures will be run upon document events. Entity: Wf_Routes

    Owner Tables Hierarchy

    • Gen_Document_Types

    Summary

    Name Type Description
    Route_Id uniqueidentifier PK
    Document_Type_Id uniqueidentifier The document type from which this route originates. Documents from this type generate sub-documents using this route.
    Process_Event nvarchar(254) Event which triggers the route. Usually the event is change of state. Every document entity may define own custom events.
    Condition_States_Bit_Mask int The system states for which to execute the specified route.
    Condition_Filter_XML nvarchar(1073741823) Contains filter condition, which the document must match in order to execute the route.
    Procedure_Name nvarchar(254) The system name of the generation procedure, which must be executed by the route.
    Destination_Document_Type_Id uniqueidentifier The type of the document, that will be generated by executing the route.
    Activation_Date date The date from which (including) the route is active. The date is matched against the document date of the generating document.
    Deactivation_Date date The date until (including) the route is active. The date is matched against the document date of the generating document. Null means the route does not have a deactivation date.
    Notes nvarchar(254)
    Active bit 1 if the route is active, otherwise 0.
    Destination_State smallint Allowed: 0, 10, 20, 30, 40, 50 0=New;10=Computer Planned;20=Human Planned;30=Released;40=Completed;50=Closed
    Read_Only bit Indicates wheather the destination document shoul be read only. 1 - the destination document is read only
    Schema_XML nvarchar(2147483647)
    Condition_Enterprise_Company_Id uniqueidentifier The enterprise company for which this route is activated.
    Destination_Enterprise_Company_Id uniqueidentifier The enterprise company in which to generate the target document.
    Allowed_Generation_Types nvarchar(1) Allowed: A, B, M Determines the possible types of the generation of the destination document: A=Auto, M=Manual, B=Both (Auto and Manual)
    Allow_Obsolete_Generation bit Allows the usage of unsupported generation procedures (marked as obsolete). This is a user override of the system prohibition of the usage of obsolete procedures.
    Connected_Party_Condition nvarchar(1) Allowed: A, C, U A - any party; C - connected party: to_party is enterprise company; U - unconnected party - not enterprise company;
    Parent_Document_Relationship_Type nvarchar(1) Allowed: S, N Determines the default relationship type between the generated document and the parent document.
    Destination_Enterprise_Company_Location_Id uniqueidentifier The enterprise company location in which to generate the target document.
    Condition_User_Status_Id uniqueidentifier The user-defined status, for which the document route is activated.
    Destination_User_Status_Id uniqueidentifier The user defined status to set to the generated document.
    Negative_Condition_Filter_Xml nvarchar(2147483647) The negative condition should NOT be matched by the document in order to execute the route.
    Row_Version timestamp
