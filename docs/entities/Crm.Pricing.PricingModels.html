    Crm.Pricing.PricingModels

    Pricing models are assigned to product groups and are used to automate creation of standard costs and prices and related price records. Entity: Crm_Pricing_Models

    Attributes

    Name Type Description
    DefaultMarginPercent decimal Default margin between standard cost and standard price. The margin is applied to the products when calculating standard price. [Required] [Default(0)]
    Id guid
    Name string The name of the pricing model. [Required] [Filter(eq;like)]
