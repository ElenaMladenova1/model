    Finance.Assets.DepreciationPlanLines Entity

    Each record contains one depreciation plan for one valuation model of one asset. Entity: Ast_Depreciation_Plan_Lines

    Attributes

    Name Type Description
    DepreciationEndDate datetime End date of the depreciation plan for this asset. [Required]
    DepreciationStartDate datetime Start date of the depreciation plan for this asset. [Required]
    Id guid
    LineNo int32 Consecutive number of the line within the depreciation plan. [Required] [Filter(eq)]
    PlanDepreciationValue Amount The part of the total amount of the asset (in the currency of the asset), which is due for depreciation. [Currency: Asset.ValuationCurrency] [Required] [Default(0)]
