Applications.Cms.NewsWebModuleArticles

    Represents an article in a news web module. Entity: Cms_News_Web_Module_Articles

    Attributes

    Name Type Description
    Id guid
    ArticleTextHtml string (nullable) The actual article text, stored as Html. Only the of the Html is stored. The language of the text is specified in Language Code.
    IsPublished boolean Specifies whether the article is published for display on web. [Required] [Default(false)]
    LanguageCode string The language code of the article text. [Required]
    PublishDate datetime The date and time when the news article should appear to be published. [Required] [Default(Now)]
