Applications.AssetManagement.ManagedAssetLocations

    Contains the locations to which the assets are assigned at various moments in time. Entity: Eam_Managed_Asset_Locations (Introduced in version 19.1.100.0)

    Attributes

    Name Type Description
    Id guid
    FromDate date The date on which the asset is assigned to the location. [Required] [Default(Now)]
    Notes string (nullable) Might contain additional notes for the exact location of the asset within the enterprise company location.
