    Finance.Cost.TemplateLevels

    Represents named hierarchy levels within a cost calculation. The hierarchy levels are used to specify cost allocation mechanisms. Entity: Cost_Template_Levels

    Name Type Description
    Id guid
    HierarchyLevel int32 The level within the hierarchy 0=Root, 1=one level below root, etc. (0..9). [Required] [Default(0)] [Filter(ge;le)]
    LevelName string The name of the hierarchy level. [Required] [Filter(like)]
