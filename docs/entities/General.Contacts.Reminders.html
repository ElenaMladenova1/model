    General.Contacts.Reminders Entity

    Reminders for activities. The reminders are alarms, which can be turned off. Each activity can have multiple reminders. Entity: Cm_Reminders

    Attributes

    Name Type Description
    Id guid
    IsDefault boolean True when this is the defult reminder for a task. The default reminder is controlled through the task, while non-default reminders are manipulated through separate form. [Required] [Default(false)] [Filter(eq)]
    IsDismissed boolean When true the reminder was dismissed by the user; the initial value is false. [Required] [Default(false)] [Filter(eq)]
    Notes string (nullable) Notes that describe this specific reminder for the task.
    ReminderTime datetime The time when the alarm should snooze to the user. [Required] [Filter(ge;le)]
