    Systems.Bpm.ProcessConnections

    Contains the connections between process elements. Part of the process model. Entity: Bpm_Process_Connections

    Attributes

    Name Type Description
    Code string Connection code, unique within the process. Used as ID for XML serialization purposes. [Required] [Default(New Guid)] [Filter(eq;like)]
    ConditionFilterXml dataaccessfilter (nullable) When not null, specifies that the flow will be followed only if the condition is matched by the current values in the process instance. [Filter(eq;like)]
    Id guid
    IsDefault boolean Denotes this flow as the default sequence flow. It is taken only when all other flows are not valid. For example, gateways usually are followed by several conditional flows and one default flow. [Required] [Default(false)] [Filter(eq)]
    Name MultilanguageString Multilanguage connection name. [Required] [Filter(eq;like)]

    References

    Name Type Description
    Process Systems.Bpm.Processes The process version to which this connection belongs. [Required] [Filter(multi eq)] [Owner]
    SourceProcessNode Systems.Bpm.ProcessNodes The element, from which the connection starts. The element should be in the same process as the connection. [Required] [Filter(multi eq)]
    TargetProcessNode Systems.Bpm.ProcessNodes The element, at which the connection ends. The element should be in the same process as the connection. [Required] [Filter(multi eq)]
