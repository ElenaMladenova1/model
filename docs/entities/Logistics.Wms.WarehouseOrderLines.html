    Logistics.Wms.WarehouseOrderLines Entity

    A planned task (operation) in a warehouse order. Entity: Wms_Warehouse_Order_Lines (Introduced in version 20.1)

    Attributes

    Name Type Description
    Id guid
    LineNo int32 Unique consecutive line number within the order. [Required]
    Notes string (nullable) Notes for this WarehouseOrderLine.
    Quantity decimal The quantity of the product, which should be operated. [Required] [Default(0)]
    TaskType string The type of the task (operation), which should be performed. MOV=Move; INB=Inbound; OUT=Outbound; INS=Inspect; PCK=Packing/Unpacking; CNT=Count. [Required]
